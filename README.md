# Todo App

A lightweight browser-based Todo app with:

- Add, complete, and remove tasks
- Local storage persistence
- Multiple visual themes (including nature themes with photo backgrounds)
- Action sound effects

## Demo

After enabling GitHub Pages, your live app URL will be:

- https://samaladyasa.github.io/Todo-app/

## Run Locally

### Option 1: Open Directly

1. Download or clone this repository.
2. Open `index.html` in any modern browser.

### Option 2: Clone with Git

```bash
git clone https://github.com/samaladyasa/Todo-app.git
cd Todo-app
```

Then open `index.html` in your browser.

## Deploy with GitHub Pages

1. Go to your repository on GitHub.
2. Open **Settings** -> **Pages**.
3. Under **Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**
4. Click **Save**.
5. Wait a minute for deployment, then open:
   - https://samaladyasa.github.io/Todo-app/

## How Data Works

- Todos are stored in your browser's local storage.
- Data is local to the device/browser profile.
- Clearing browser site data will remove saved todos.
